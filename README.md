# Law & Learning

**Making Law Learnable.**

Law & Learning applies cognitive science to legal education through practical study tools, open resources and evidence-based learning design. This repository contains the source for [lawandlearning.com](https://lawandlearning.com).

## What's here

- `index.html` — Homepage
- `/court-observation-tool/` — Interactive court-observation tool
- `/gdpr-study-map/` — GDPR study map
- `/partials/` — Shared HTML partials
- `/assets/`, `favicon.svg` — Site assets and identity
- `*.pdf` — Printable study and observation templates
- `legal.html` — Use & Privacy
- `404.html`, `sitemap.xml`, `robots.txt` — Site infrastructure
- `WORKFLOW.md` — Internal build notes
